Компоновщики элементов
Для изменения типа компоновщика элементов используется метод setLayout() для нашей формы.
setLayout(new FlowLayout());

BorderLayout - позволяет располагать элементы по сторонам света.
BorderLayout.NORTH, BorderLayout.SOUTH, BorderLayout.WEST, BorderLayout.EAST, BorderLayout.CENTER

BoxLayout - Располагает элементы в строку или столбец в зависимости от используемой константы:
BoxLayout.Y_AXIS - в столбец
BoxLayout.X_AXIS - в строку.
setLayout(new BoxLayout(getContentPane(), BoxLayout.X_AXIS));
При создании BoxLayout в конструктор необходимо передать ссылку на контейенр, в котором он будет работать. Для получения
у формы ссылки на внутренний контейнер используется метод getContentPane().

FlowLayout - Располагает элементы в одну строку, когда ширины строки становится недостаточно, переносит новые элементы
на следующую.
setLayout(new FlowLayout());

GridLayout - Элементы управления раскладываются по ячейкам таблицы, для которой задано количество строки и столбцов.
Высота строк и ширина столбцов зависит от их количества и размеров формы, то есть вручную задать значения этих свойств
нельзя.
setLayout(new GridLayout(3, 3));

Отключение компоновщика элементов - Возможен сценарий ручной расстановки элементов путем указания их абсолютных
координат. Для этого необходимо указать setLayout(null) и для каждого элемента указать его координаты и размеры с
помощью метода setBounds().

setLayout(null);
JTextField field = new JTextField();
field.setBounds(20, 20, 120, 32);
add(field);
JButton button = new JButton("Нажми сюда");
button.setBounds(20, 60, 120, 32);
add(button);

При таком подходе вы полностью можете управлять тем, где и как будут расположены элементы, но на разных операционных
системах интерфейс будет выглядеть по-разному, и не всегда так как вы задумывали.

Базовые элементы управления
JFrame – окно приложения;
JButton – кнопка;
JLabel – надпись;
JTextField – однострочное текстовое поле;
JTextArea – многострочное текстовое поле;
JScrollPane – контейнер для пролистывания контента;
JMenuBar – верхнее меню программы;
JTable – таблица;
JRadioButton – RadioButton;
JCheckBox – CheckBox.
